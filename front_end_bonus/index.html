ETL Video Game Data Dashboard

The purpose of this project was to extract, transform, and load into a database, data on Video Game Sales. We compiled as much data as we could find related to video game sales, cleaned it up, and loaded it into a SQL database. We then analyzed the data and created a dashboard to showcase our efforts.
